# BloodConnect ‚Äì Online Blood Donation System

**BloodConnect** is a full-featured PHP-based web application designed to streamline the process of blood donation and management. This system allows users to register, login, request blood, and provides an admin panel to manage all donation requests efficiently. It demonstrates skills in **PHP, MySQL, HTML, CSS, and web development best practices**.
